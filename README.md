### 적용 기술

> - 언어 : HTML5 , CSS3 , Java , Typescript
> - 프레임워크 / 라이브러리 : 
React , Spring , Redux , Redux-Saga, Next, Query DSL, ApexChart, BootStrap
> - 데이터베이스 : PostgresSQL
> - AWS : EC2 , RDS,  S3,  CloudFront
> - 미들웨어 : RabbitMQ
